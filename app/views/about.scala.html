About RDFShape

RDFShape can be used as an online RDF syntax checker. It can be used can check syntax and convert between different RDF syntaxes. It can also be used as an RDF validator based on Shapes: both ShEx (Shape Expressions) and SHACL are supported.

Developed by: Jose Emilio Labra Gayo, WESO Research Group

Source code: Github project
