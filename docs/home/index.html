RDFShape API Web Docs

0.2.12​

The following docs contain guides on the usage and deployment of RDFShape API. Therefore, this may be useful:

  • From a back-end perspective: you want to consume the API in a machine-readable format.
  • If you are willing to build your own API client to use RDFShape for your particular use case.

If you are willing to consume the API graphically or use it as a playground with a human-readable format, refer to our own RDFShape Client.

WESO members may access additional documentation here.
