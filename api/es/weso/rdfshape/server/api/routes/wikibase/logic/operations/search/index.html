  4.  sealed case class WikibaseSearchSchema extends WikibaseSearchOperation with Product with Serializable

    A WikibaseSearchOperation searching for schemas in a wikibase instance.

    A WikibaseSearchOperation searching for schemas in a wikibase instance. Unlike other search operations, there is no straightforward method to get schemas with their labels and descriptions, so this is our custom solution. 1- Resorts to https://www.wikidata.org/w/api.php?action=help&modules=query%2Bsearch to get the pageIds of the schemas that match the query (by name) 2- Resorts to https://www.wikidata.org/w/api.php?action=help&modules=query to query the pageIds of each schema and get more detailed information about them 3- For each schema, adds the URL to get their ShEx content so that clients can easily use them
