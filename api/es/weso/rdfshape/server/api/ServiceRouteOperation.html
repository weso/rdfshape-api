ServiceRouteOperation

trait ServiceRouteOperation[C] extends AnyRef

This trait defines an operation performed by a route in a service and the way to decode the incoming data into the required object

All errors occurred during object instantiation will be mapped to DecodingErrors

C

Type into which the client data will be decoded for usage in this route
